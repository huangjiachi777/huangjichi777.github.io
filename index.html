CASE 02
品牌策划赛道国奖 — 双品牌策划案
品牌策划 营销方案
+

成果

产出
2份完整品牌策划书 + 路演PPT

独立完成两个品牌的完整策划案：旭日升茶饮（怀旧复兴） + 隆延茶铺中药奶茶（国潮跨界）

策划案一：旭日升茶饮 — 怀旧复兴品牌策划

  • 市场分析：2023年中国茶饮料市场突破 2,000 亿元，无糖茶饮市场份额超 25%
  • 品牌定位：中国第一款冰茶品牌，承载 80/90 后集体记忆，主打「复古+创新」双线策略
  • 消费者洞察：18-35 岁 Z 世代占 60%，80/90 后怀旧群体占 30%，健康化+功能化+个性化是核心趋势
  • 营销方案：复古场景化营销——与大白兔、北冰洋联名「童年记忆礼盒」；「重返90年代」快闪店；校园怀旧巡演音乐节
  • 渠道策略：传统商超红白复古堆头 + 便利店冰柜黄金位 + 高铁站自动贩卖机 + 社区团购「家庭怀旧套餐」

策划案二：隆延茶铺 — 中药奶茶国潮跨界

  • SWOT 分析：健康属性（优势）vs 中药味道接受度（劣势）；健康风潮（机遇）vs 传统奶茶竞争（威胁）
  • 产品创新：二十四节气限定系列——雨水·茯苓普洱、立夏·黄芪金盏、白露·白芍玫瑰、冬至·当归甜韵
  • IP 打造：「茶医仙子」人物形象+中医服饰+时尚元素融合，品牌周边（书签、手提包、杯子、明信片）
  • 营销策略：小杯试饮体验+养生知识讲座+限定礼盒，消除消费者对中药口味的认知偏差
  • 风险管控：员工培训标准化、市场趋势跟踪、品牌建设持续投入，总预算 35.5 万元

品牌策划赛道国家一等奖，两份完整策划书覆盖市场分析→品牌定位→营销方案→风险管控全流程

CASE 03
美的集团商业计划书与市场分析

90页完整商业计划书
覆盖
PEST / SWOT / 五力 / 财务预测

为美的集团完成全维度商业分析：宏观环境→行业竞争→财务分析→战略规划→估值预测

背景

以美的集团（000333）为标的，分析其在家电行业的竞争力并制定战略建议。美的覆盖智能家居、楼宇科技、工业技术、机器人与自动化五大业务板块，年营收超 3,200 亿元。

策略与执行

  • 宏观环境分析（PESTEL）：梳理双碳政策、家电下乡、碳关税、汇率波动等关键外部变量
  • 行业竞争分析（波特五力）：识别家电行业集中度、替代品威胁、新进入者壁垒
  • SWOT 分析：美的核心优势（33个研发中心、400+子公司、覆盖200+国家）vs 行业挑战
  • 财务分析：资本结构、偿债能力、盈利能力、营运能力四维度量化分析
  • 财务预测：三年量化目标 + 分业务板块预测 + 估值与投资回报测算

商业计划书共9章 90页，系统性展示了从行业研究到财务预测的完整分析链路

思考路径

  • 建立标准化的活动执行 Checklist，从场地确认、设备调试到签到流程，减少遗漏
  • 针对不同规模企业定制接待方案，500强企业 vs 本地中小企业的资源分配策略不同
  • 活动后的复盘机制：统计到场率、学生满意度、企业反馈，用于迭代下一场

角色
市场数据分析 & 财务风控
成果
国家二等奖（Top 800+院校中）
核心指标

背景

在数智化沙盘中模拟企业全流程运营，800+ 本科院校队伍同台竞争，需要在 8 期决策中持续优化策略。

策略与执行

  • 通过动态市场趋势分析优化产品布局，每期根据竞争对手策略调整定价和产能
  • 与团队协作完成 8 期企业运营决策，负责市场数据分析与财务风险管控
  • 最终实现模拟营收超行业平均水平 27%，库存周转率提升 40%

库存周转率提升 40%，营收超行业平均 27%

思考路径

  • 每期决策前先分析上期数据——哪个产品线利润最高？竞争对手的价格策略是什么？
  • 库存周转率是关键杠杆：降低滞销产品产能、增加高毛利产品投入，实现资金效率最大化
  • 不追求单期最优，而是 8 期全局最优——有时需要牺牲短期利润换取市场份额
CASE 06
智能电吹管 — 天使轮融资与风控
策略分析 风险管理
+
角色
项目核心成员（财务 & 风控）
级别
国家级立项
工具
市场法+收益法双验证

背景

大学生创业项目进入天使轮融资阶段，需要完整的财务模型、合规方案和风险应对策略来打动投资人。

策略与执行

  • 独立完成天使轮融资全流程财务工作：估值测算（市场法+收益法双重验证）、股权架构设计
  • 建立月度滚动预算机制，完成单因素+双因素敏感性分析，识别供应链、市场、现金流三大核心风险
  • 负责财务合规与税务筹划，充分利用大学生创业税收优惠政策
  • 成功对接投资人与校企资源

用市场法 + 收益法双重验证估值，制定三大核心风险应对措施

思考路径

  • 投资人最关心两件事：估值是否合理、风险是否可控——围绕这两个核心设计材料
  • 敏感性分析不仅展示了风险意识，更展示了「我们已经想过最坏情况」的准备度
  • 税务筹划不是财务技巧，而是创业项目的真实成本优势——把它放在投资回报测算里，变成亮点

背景

部门预算管理流程分散、执行跟踪不及时，传统 Excel 共享方式导致信息滞后和版本混乱。

策略与执行

  • 独立完成 100+ 份会计凭证整理归档，确保附件完整率 100%

用低代码工具将分散的 Excel 流程升级为实时协同的数字化看板

思考路径

  • 先花时间理解现有流程的痛点——不是「我要搭一个表格」，而是「什么信息在什么时候需要被谁看到」

  • 搭建完成后主动向主管演示，并做了使用指南文档，降低团队迁移成本
